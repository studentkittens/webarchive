Logging¶

Overview¶

Removed by Python Logger

Logs all Warning messages. All messages are written to Log File. You can set a loglevel. This Level is only set for shell written logs. All messages obove loglevel are written to shell.

Additional Notes:

Possible Log levels: - debug - info - warning - error - critical

Implementation¶

Setting Severity:

archive.logger.loghandler.severity(value)[source]¶

Set Loglevel. Possible level’s: debug, info, warning, error and critical.

Setting Logfile:

archive.logger.loghandler.file(path)[source]¶

Set Path for Logfile.

Acquiring All Logs:

archive.logger.loghandler.log_read()[source]¶

Prints all Log Messages to shell
