Extracting data from crawled content¶

Overview¶

One Requirement was to extract the title from HTML Documents. There is no Plugin-System as for the Filters, but one may write an extractor function that will be called for a specific mimeType and register it like this:

# In crawler.extractor.py
EXTRACTORS = {
      "text/html": extract_html
      "application/pdf": extract_pdf
}

Implementation¶

Title extractor module which makes possible to extract title names from various content files. Currently only text/html files are supported.

archive.crawler.extractor.extract_html(file_path)[source]¶

Title extractor for text/html files.

File_path :path to html file
Returns:extracted title or empty string if no title is found
archive.crawler.extractor.get_title(file_path, mime)[source]¶

Extractor ‘interface’ to delegate file by mime type

File_path :path to content file
Mime :mime type of that file to determinate extractor
Returns:extracted title as string if extraction is successful else an empty string will be returned
